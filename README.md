# FREQEE – Music Portfolio

This repository contains the source code for the public composition portfolio:

https://chamber.freq.ee

The site presents selected works by FREQEE, including compositions, interpretations, and experimental sound systems.  
It is implemented as a lightweight static site with a custom JavaScript audio catalogue.
